# Анализ поступления в вуз (4-дневный мониторинг)

Данный проект представляет собой веб-приложение на базе фреймворка Django, предназначенное для анализа динамики подачи заявлений и зачисления абитуриентов в вуз в течение первых четырех дней приемной кампании.

Система позволяет агрегировать данные о поступающих, визуализировать конкурсную ситуацию по специальностям и прогнозировать проходные баллы на основе статистики за короткий промежуток времени.

## Описание проекта

Приемная кампания в вузах часто сопровождается высоким темпом изменений конкурсных списков. Особенно важными являются первые дни после старта подачи документов. Этот проект создан для решения задачи оперативного мониторинга и анализа этих данных.

**Основная цель:** Предоставить сотрудникам приемной комиссии или самим абитуриентам инструмент для отслеживания текущей ситуации (баллы, позиции в рейтинге, шансы на зачисление) на раннем этапе поступления.

## Функциональные возможности

*   **Импорт данных:** Загрузка данных об абитуриентах (баллы ЕГЭ, индивидуальные достижения, приоритеты, направления подачи документов) из CSV файлов.
*   **Динамика по дням:** Отслеживание изменения количества поданных заявлений по каждой специальности за каждый из 4-х дней.
*   **Рейтинговые списки:** Построение ранжированных списков абитуриентов по специальностям с подсчетом суммы баллов.
*   **Аналитика:** Расчет минимального проходного балла на текущий момент и прогноз проходного балла на следующий день.
*   **Визуализация:** Графики, показывающие конкурс.

## Технологии

*   **Backend:** Python 3.10+, Django 4.x
*   **База данных:** SQLite
*   **Фронтенд:** HTML, CSS, JavaScript

## Установка и запуск

Следуйте этим инструкциям, чтобы развернуть проект локально.

### Предварительные требования
- Установленный `Python` (версия 3.10 или выше)
- Установленный `Git` 
- Менеджер пакетов `pip`

### Шаги по установке

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/DVGroin/projectForPredporofOlimp
    cd путь-по-которому-вы-склонировали
    
    ```

2.  **Создайте и активируйте виртуальное окружение:**
    ```bash
    # Для Windows
    python -m venv venv
    venv\Scripts\activate

    # Для Windows через cmd
    python -m venv venv
    venv\Scripts\activate.bat

    # Для Windows через powershell
    python -m venv venv
    venv\Scripts\Activate.ps1

    # Для Linux/macOS
    python3 -m venv venv
    source venv/bin/activate
    ```

3.  **Установите зависимости:**
    ```bash
    pip install -r requirements.txt
    ```

4.  **Примените миграции:**
    ```bash
    python manage.py makemigrations
    python manage.py migrate
    ```

5.  **Запустите сервер разработки:**
    ```bash
    python manage.py runserver
    ```

6.  **Откройте браузер и перейдите по адресу, который был выведен в консоль, обычно это:**
    `http://127.0.0.1:8000/`
